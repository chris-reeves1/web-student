CSS:

Three main hooks:
    element selectors (p, h1)
        - global, normal style, reseting behaviour.
        - Production - be mindful of changes when all are affected.
        - be cautious dont overuse.
    class selectors(.intro, .btn)
        - reusable styling hook. 
        - real world use these.
        - flexible.
    ID selectors(#main-title)
     - unique one per page, inflexible. 
     - highest priority
     - overhead in terms of maintainence. 

Design: maintainabilty, scalability, efficiancy. 

Cascade:
    inline -> ID -> class -> element. 



Font fallbacks:
- goes through the list if font isnt found.  

line-height:
- use modern unitless approach scales with font size well. 

body block:
- defaults for everything in body. 
- prevents randomness from browser.

    body {
        font-family: Arial, Helvetica, sans-serif;
        background: var(--bg);
        color: var(--text);
        margin: 0;
    }

vars:
- Store css vars
- Enforce consistancy.
- makes wide changes easier. 
- reduces duplication.
- switch between themes dark/light

    :root {
        --text: #222222;
        --muted: aliceblue;
        --card: red;
        --bg: aliceblue;
        --tbc: blueviolet;
    }

Box model:

Everything is a box!
padding = internal space. 


html - semantics (head, nav, main, footer), headers, tables , forms etc
     - add hooks. classes for components for styling. id's for js. 

css - globals (defaults), major layout (flexbox).
    - styling components - cards, forms, buttons, inputs.
    - state - focus, hover, highlights.
